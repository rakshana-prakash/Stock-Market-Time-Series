# Stock Market Time Series Analysis

## 📌 Objective
To analyze stock price trends using time series techniques.

## 📊 Dataset
Contains stock data with:
- Open, High, Low, Close prices
- Volume
- Date (time index)

## ⚙️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📈 Analysis Performed
- Trend Analysis
- Moving Averages
- Autocorrelation
- Data Visualization

## 📌 Result
Identified trends, volatility, and time dependency in stock prices.

## 🚀 Future Scope
- Advanced models (ARIMA, LSTM)
- Real-time data integration
